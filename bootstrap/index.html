Software Development That Delivers

Software development, broken down to it's simplest explanation, is just a set of instructions that the computer can complete, one instruction at a time, until the desired result is accomplished.

So one would think that it wouldn't be that hard to create software that accomplishes the intended goal. Is the first thought that runs through the mind of a new software developer, "this is so much easier than I thought it was going to be?"

It's Like a New Language

When you first hear a foreign language, can you understand the message that is trying to be communicated? Most software developers experience the same kind of feeling when attempting to write code for their very first program.

The Power of a Translator

My job as a software developer is to take the message of my client and translate that message into a set of instructions that the computer will be able to use to communicate that message both accurately and efficiently.

Sounds easy enough.

Adding Value to the Bottom Line

This site will provide an "over my shoulder" viewpoint of my journey to becoming a software developer that brings value to the bottom line of those who use my set of computer instructions.

Along the way, I will try and pass on any helpful tips that I pick up on my journey to becoming a junior software developer.
